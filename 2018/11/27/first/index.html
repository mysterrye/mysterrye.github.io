hexo+gitpage小结

准备：安装node，git
1建立目录文件夹 安装hexo 命令 sudo npm install -g hexo 初始化 hexo init

2生成查看页面 生成静态页面 hexo generate

3 hexo server #Hexo 会监视文件变动并自动更新，您无须重启服务器 浏览器输入地址动态查看页面

4配置github 申请账号，master下创建username.github.io仓库，配置SSH key可以不用每次输密码

打开_config.yml建立git关联
deploy:

type: git

repo: https://github.com/leopardpan/leopardpan.github.io.git

branch: master

然后执行命令：

npm install hexo-deployer-git –save

5最后 配置命令 hexo deploy 打开username.github.io查看初始页面
